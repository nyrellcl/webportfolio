Projects

As a self-taught front-end engineer, it's important for me to learn by doing. Creating projects have become a focal point in my journey.

Doing so helps to refine my design process and identify habits that can either be beneficial or something that requires improvement.

Photo Portfolio site

This is my first self-led project I created a few months after I started learning HTML and CSS.

I wanted to demonstrate to myself how much I've learned and how I think as I create projects without outside direction.

This was very good practice for me as it allowed me discipline in completing what I start and leading myself to do so.

Huddle Landing Page

Completing projects are important towards my development as an engineer.

This landing page gave me valuable insight on the typography, call to action designs, and fluidity that goes into creating a product that is intended for a high volume of users to visit.

Sunnyside site

This was also another project I replicated for more practice.

Creating different landing pages with entirely different designs exposes me to a variety of areas of design and development.

It's important to be exposed to different projects as it allows me more unique ways of thinking.

Lani site

Along with the first project, I created this one for fun. It became clear to me how important it is to have fun while striving for a career in this field. So I created this page using my friend's artworks.

This was also the first project where I incorporated a slideshow feature, where it allows the user to view multiple images by scrolling horizontally for desktop and swiping for mobile.

Immersiva

This was a mock website I created referencing an actual company's website. I wanted to see how much I can change their website, while still keeping their brand identity intact.

This project was very exciting to me. I was able to reference a code snippet for a parallax effect for when the user scrolls down.

The goal of this project was to create a complete homepage. Complete enough that I could present it to their designer with complete research and evidence to back up my reasonings for creating this design.

Form Validation

This project I made following a tutorial. I'm still grasping Javascript and I found it extremely helpful for me to follow along code by code on how to make a project using a language I'm not yet proficient in.

While following this tutorial I realized I should stick to finding tutorials involving Javascript and create it. The most important thing to keep in mind was understanding what I was actually programming.

This project provided a great breakthrough to how I should be practicing Javascript and implementing the concepts.
